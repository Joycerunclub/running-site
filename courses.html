Course 01
初跑班

適合沒有跑步習慣、想培養運動習慣的你。從最基礎開始，讓跑步變成生活中「有意思」的事。

  • ✓ 從零開始，循序漸進建立跑步習慣
  • ✓ 跑姿基礎教學，減少受傷風險
  • ✓ 暖身、跑步、緩和的完整訓練
  • ✓ 與同學互相鼓勵，一起成長
  • ⏱️ 每堂課 90 分鐘

賽事訓練班

有跑步基礎、想挑戰比賽成績的你。週期化賽事訓練，幫你突破配速瓶頸，達到目標完賽時間。

  • ✓ 週期化賽事訓練計劃
  • ✓ VDOT 能力評估與配速規劃
  • ✓ 速度訓練、長跑、節奏跑
  • ✓ 比賽策略與心態調整
  • ⏱️ 每堂課 90 分鐘

跑者肌力班

專為跑者設計的肌力訓練課程。強化核心與下肢力量，讓你跑得更穩、更有效率，同時降低受傷機率。

  • ✓ 跑者專屬肌力訓練動作
  • ✓ 核心穩定性強化訓練
  • ✓ 臀部、大腿、小腿強化
